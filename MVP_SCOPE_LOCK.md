# MVP Feature Scope Lock

## 🟢 MVP (Must Have)
1.  **Personalized Roadmaps**: Generate tracks based on experience/role.
2.  **Skill Graph**: Structured curriculum (Track > Phase > Skill > Milestone).
3.  **Progress Tracking**: Mark milestones complete, visualize progress %.
4.  **Milestone Validation**: Basic Q&A to verify learning.
5.  **Admin CMS**: Internal tools to manage roadmap content.
6.  **User Auth**: Secure sign-up/login (Google/Email).

## 🟡 Post-MVP (Nice to Have - v1.1)
1.  **Streaks & Gamification**: Daily activity tracking (Schema exists, UI does not).
2.  **Social Profiles**: Publicly shareable roadmaps.
3.  **Payment Gating**: Stripe integration for premium tracks.
4.  **AI Tutor**: Chatbot for questions (Integration logic exists but unused).

## 🔴 Remove / Deprecated
1.  **Legacy DSA**: Old "Sheet" and "Problem" lists. (Code exists but hidden).
2.  **Community / Forum**: Removed routes.

## MVP Definition
> **"A personalized, tracked learning path for engineers that validates understanding at every milestone."**
