# akashcodeofficial.com

This is the personal brand website for Akashdeep Thanda.

## Current Focus
Website revamp — adding DSA Sheet with auth, interactive roadmaps,
blog section, and improved landing page.

## PRD
Full requirements documented in akashcodeofficial_PRD_v1.docx

## Key Priorities
1. Never break existing pages while adding new features
2. Keep bundle size under 300KB gzipped
3. All new components must be mobile-responsive
4. Maintain dark mode / neon green design system throughout

## Before Any Change
- Check if the component already exists before creating a new one
- Run npm run dev and verify it works before committing
